How to Test

Open Developer Tools

Press F12 or right-click and select "Inspect", then go to the Console tab

Click the Chat Widget

The widget should appear in the bottom-right corner. Click to open it.

Look for Console Logs

You should see: [Hi Chroney Widget] Sending parent page URL to iframe followed by the current page URL

Create a Test Lead

Chat with the widget and provide your contact details (name, email, phone)

Check Server Logs

Look for [LeadSquared] Creating lead with Mx_Website_Campaign in the fields
